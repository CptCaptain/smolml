## Scope
- `enwik8` download + prep (byte-level), deterministic split, tiny bundled sample for offline tests.
- **Honest, documented FLOP counter** (forward+backward), shared by all models — the critical
  correctness surface.
- Training loop to a **fixed FLOP budget**; fixed seed; per-run JSONL log
  (wallclock, step, cumulative_flops, train_loss, val_bpb).
- **Model registry / interface** so future candidates plug in with no harness changes.
- Transformer **baseline** (pre-norm, RMSNorm, RoPE, causal), configurable small sizes.
- **Leaderboard**: aggregate runs → table + bpb-vs-FLOPs plot.
- Device auto-detect cuda>mps>cpu.

## Acceptance criteria
- `uvx ruff format --check`, `uvx ruff check`, `uv run pytest` all green (paste outputs).
- Unit tests: FLOP-counter vs. hand-computed values; bpb correctness; tiny offline end-to-end
  smoke run that yields a leaderboard row + writes a plot file.
- `docs/harness.md`: how to run, add a candidate, regenerate the leaderboard.
- Per `AGENTS.md`: add/update `docs/learning/` for any concept introduced.
